## BUG-079: 生产环境手册页面返回 404

| 字段 | 内容 |
|------|------|
| Bug ID | BUG-079 |
| 发现版本 | v0.9.0 |
| 发现阶段 | 回归测试 |
| 发现日期 | 2026-03-07 |
| 优先级 | P2 |
| 状态 | 待修复 |

### 问题描述

点击"帮助"按钮后，页面返回 404 错误。测试环境 (8081) 正常工作，但生产环境 (8080) 返回 404。

### 复现步骤

1. 打开生产环境 http://localhost:8080
2. 点击右上角"❓ 帮助"按钮
3. 页面显示 404 Not Found

### 预期行为

帮助手册正常显示。

### 实际行为

页面返回 404 错误。

### 根因分析

测试环境已添加 `/manual.md` 路由 (`dev/app/__init__.py`)，但生产环境未重新部署。

### 修复方案

重新部署生产环境 (8080)，或手动在生产环境添加以下路由：

```python
@app.route('/manual.md')
def serve_manual():
    return send_from_directory(base_dir, 'manual.md')
```

### 修复状态

- [x] 开发环境修复 (dev/)
- [ ] 生产环境部署 (计划在下个版本发布时部署)

### 备注

生产环境部署计划: v0.9.1 或后续版本
