## 2. T2: Tabs 样式迁移 - 审查结果

### ⚠️ 发现问题

| 检查项 | 状态 | 说明 |
|--------|------|------|
| index.html 中 Tabs 样式 | ❌ 仍存在 | L16-19 仍有定义 |
| design-system.css 中定义 | ✅ 符合规格 | 使用底部指示器 |

#### 问题详情

**index.html L16-19 (旧样式)**:
```css
.tabs { background: #e0e0e0; border-bottom: 2px solid #2170bb; }
.tab { background: #e0e0e0; }
.tab.active { background: #2170bb; color: white; }  /* 蓝色背景 ❌ */
```

**design-system.css L592-597 (新样式)**:
```css
.tab.active {
  background: transparent;  /* 透明背景 ✅ */
  color: var(--color-primary);  /* 紫色文字 ✅ */
  border-bottom: 2px solid var(--color-primary);  /* 底部指示器 ✅ */
}
```

### 影响

由于 index.html 中的样式仍在，会覆盖 design-system.css 中的新样式，导致：
- Tab 激活状态仍显示蓝色背景
- 无法实现规格要求的"底部指示器"效果

### 修复建议

删除 index.html L16-19：
```html
<!-- 需要删除 -->
<style>
  .tabs { ... }
  .tab { ... }
  .tab.active { ... }
  .tab:hover:not(.active) { ... }
</style>
```
