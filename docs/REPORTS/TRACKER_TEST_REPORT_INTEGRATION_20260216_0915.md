### 问题分析

**TC-001 失败原因**: `browser.newContext: Target page, context or browser has been closed`

这是一个测试环境问题，不是代码功能问题。Browser context 在测试执行过程中意外关闭，可能由于：
1. 测试执行时间过长
2. 内存不足
3. Firefox 进程异常

后续测试 (TC-002 ~ TC-011) 均正常执行并通过，证明实际功能没有问题。
