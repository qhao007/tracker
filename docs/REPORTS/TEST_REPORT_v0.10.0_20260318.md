## 4. 测试代码修复记录

> 本章节记录测试代码问题的修复

### 4.1 CP 创建字段修复

| 问题 | 修复前 | 修复后 |
|------|--------|--------|
| API 字段名称错误 | 使用 `name` 和 `description` 字段 | 使用 `feature` 和 `cover_point` 字段 |

**原因**: 创建 CP 的 API 需要 `cover_point` 字段而非 `name`

### 4.2 测试数据唯一性问题

| 问题 | 修复方案 |
|------|----------|
| 项目名称重复导致创建失败 | 使用 UUID 生成唯一项目名称 |

**原因**: 多个测试用例并发/快速执行时，时间戳可能相同导致项目名称冲突
