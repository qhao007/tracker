## 1. 功能概述

### 1.1 功能描述

用户手册功能允许用户通过点击"帮助"按钮查看系统使用文档。

### 1.2 对应规格

| 规格文档 | 相关章节 |
|----------|----------|
| `tracker_OVERALL_SPECS.md` | 2.3 用户手册：在线查看使用帮助 |
| `tracker_SPECS_v0.7.1.md` | 5. 需求 4: 用户手册 |

### 1.3 规格要求

| 需求项 | 要求 |
|--------|------|
| HELP-REQ-001 | 帮助按钮显示在首页顶部工具栏 |
| HELP-REQ-002 | 点击后在新标签页打开手册 |
| HELP-REQ-003 | 手册内容为 Markdown 渲染 |
| HELP-REQ-004 | 手册包含完整的系统使用说明 |
