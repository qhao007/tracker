# 阶段1：测试覆盖分析快速参考

## 核心任务

1. **审查测试覆盖增强计划**
   - 读取: `docs/PLANS/test_coverage_enhancement_plan_v{version}.md`
   - 分析 COV-XXX 问题

2. **提取测试用例清单**
   - API 测试需求
   - UI 测试需求
   - 手工测试需求

3. **生成详细开发计划**

## 输出格式

```markdown
## 测试覆盖分析报告

### 分析摘要
| 测试类型 | 数量 | 优先级 |
|----------|------|--------|
| API 测试 | N | 高/中/低 |
| UI 测试 | N | 高/中/低 |
| 手工测试 | N | 高/中/低 |

### 详细开发计划
[每个测试用例的详细计划]
```

## 关键点

- 测试用例ID格式: `API-COV-XXX`, `UI-COV-XXX`, `MANUAL-COV-XXX`
- 必须识别所有需要补充的测试
- 区分优先级
